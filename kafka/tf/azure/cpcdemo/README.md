# Confluent Private Cloud Gateway Demo

## Steps to Deploy

1. Create 2 AKS clusters one for each CP cluster
2. Deploy 2 CP Cluster one in each AKS cluster
3. Create 1 AKS cluster for the gateway
4. Deploy the gateway in the gateway AKS cluster
5. Create 1 AKS cluster for client
6. Run multiple clients from the client AKS cluster
7. Observe the gateway logs and metrics
